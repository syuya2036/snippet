# 二分探索

## lower_bound

```rust
fn lower_bound<T: Ord>(A: &Vec<T>, v: &T) -> usize {
    let mut l = 0;
    let mut r = A.len();
    let mut mid;
    while r - l > 1 {
       mid = (l + r) / 2;
       if A[mid] >= *v {
            r = mid
       } else {
           l = mid
       }
    }
    r
}
```
